## 1. Thông tin:

* Mod: Hnah Joker
* Hỗ trợ:

3. Danh sách Layout

    - QWERTY 0 (Default)
    - MACRO 1
    - NUMPAD 2
    - COLEMAX 3
    - FUNC 4
    - LED 5
    - MOUSE 6
    - Change Layer 7
